# VisualShader-IdMap

VisualShader IdMap masking Node

Inputs:
	
	* IdMap Texture : IdMap Texture
	* Colorkey : Color to extract mask from IdMap Texture
	* Tolerance : Color values tolerance (Default = 0.05)
	
Output:
	
	* Mask : Float value of the resultant mask
